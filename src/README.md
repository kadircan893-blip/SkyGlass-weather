SkyGlass Weather is a modern weather application that displays the current weather and a 2-day forecast for cities in Turkey.
The app includes dynamic canvas-based visual effects and ambient sound effects that change according to the weather.

Features

---

-Current weather information
-2-day weather forecast
-City search (Turkey â€“ 81 provinces)
-Dynamic weather visuals (rain, snow, sun, clouds)
-Weather-based sound effects
-Modern glass-style UI
-Responsive design

Technologies

---

-React (Vite)
-TypeScript
-Tailwind CSS
-Canvas API
-WeatherAPI

Installation

1. Install dependencies:
   npm install

2. Create .env file in root folder:
   VITE_WEATHER_API_KEY=your_api_key_here

3. Run project:
   npm run dev
